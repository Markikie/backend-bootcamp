# Backend Engineer Zero → Strong Junior

A 3-month backend engineering bootcamp focused on building strong fundamentals
and reducing dependency on AI-generated code.

## Goal

สามารถสร้าง Backend API ด้วย Go ได้ด้วยตัวเอง และอธิบายโค้ดที่เขียนได้ทุกส่วน

## Main Stack

- Go
- PostgreSQL
- Git
- Docker
- REST API

## Learning Method

แต่ละหัวข้อใช้ระบบ Checkpoint:

1. Learn
2. Quiz
3. Coding Assignment
4. Code Review
5. Challenge
6. Pass

## AI Usage Policy

AI ใช้สำหรับ:

- อธิบายแนวคิด
- ให้คำใบ้
- Review โค้ด
- ช่วยวิเคราะห์ Error

AI ไม่ใช้สำหรับ:

- เขียน Assignment ทั้งหมด
- เขียน Feature ให้ทั้งก้อน
- Copy โค้ดโดยไม่เข้าใจ
